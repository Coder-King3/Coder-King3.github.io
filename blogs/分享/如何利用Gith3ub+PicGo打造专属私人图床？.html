如何利用Github+PicGo打造属于自己的私人图床？

CoderKing2022-09-11 00:00:00分享GitHub

# 1. 注册一个Github账号并下载PicGo

Github：https://github.com/，有的话直接登录看下一步。

PicGo：https://github.com/Molunerfinn/PicGo/releases ，自行选择需要的版本下载。

# 2. 配置Github

# 2.1 创建一个仓库/New repository，当做存储图片的图库

注意：仓库必须是公开的，否则图片不能正常访问。

# 2.2 设置一个Token用于PicGo访问Github

(1)找到令牌菜单

(2)创建令牌

创建完成后注意保存令牌（令牌只显示一次，请妥善保存）

# 3. PicGo配置

# 3.1 配置github图床

设定仓库名：你创建的github仓库名（[username]/[仓库名]）

设定分支：设置需要上传的分支（需与设定自定义域名分支保持一致）

设定ToKen：添加已创建的ToKen

设定储存路径：设置储存的路径（不能为空）

设定自定义域名：https://cdn.staticaly.com/gh/[username]/[仓库名]@master

原本github的自定义域名应该是：https://raw.githubusercontent.com/[username]/[仓库名]，但是这种方式访问图片巨慢，所以用到了cdn加速staticaly。
